Задачи курсовой работы:

\begin{enumerate}
    \item Содержательная и математическая постановка задачи. В этом разделе будет рассмотрена теоретическая составляющая приближенного решения TSP на основе минимального остовного дерева (MST), его принципы работы и особенности. Будет дано математическое описание алгоритма и доказательство минимальной точности аппроксимации, достигаемой этим методом;
    \item Реализация алгоритма. В этом разделе будет приведена реализация алгоритма средствами языка ``Rust'';
    \item Реализация программы визуализации. В этом разделе будет преведена реализация программного средства визуализации работы алгоритма.
\end{enumerate}

Каждый из этих разделов представляет собой важную составляющую курсовой работы и является необходимым для достижения целей работы.
